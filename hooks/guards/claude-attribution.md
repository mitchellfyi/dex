---
name: block-claude-attribution
enabled: true
event: bash
pattern: (?s)(?=.*\b(?:gh\s+pr\s+(?:create|edit|comment)|git\s+commit)\b)(?=.*(?:Generated with Claude Code|Generated by Claude(?: Code)?|Created by Claude(?: Code)?|Authored(?: by| with) Claude(?: Code)?|Co-Authored-By:\s*Claude))
action: block
case_sensitive: false
---

BLOCKED: Claude attribution detected in a PR or commit command.

Doyaken owns lifecycle attribution. Use `Generated by Doyaken` in PR bodies when attribution is needed, and use `Co-Authored-By: Doyaken <noreply@doyaken.ai>` for commit trailers. Do not add Claude Code generated-by footers or Claude co-author trailers.
